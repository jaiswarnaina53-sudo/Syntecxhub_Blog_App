# 📝 Blog Application — MERN Stack

A full stack blog platform built with the MERN stack where users can register, log in, and manage their blog posts.

## 🚀 Features

- User Registration & Login with JWT Authentication
- Create, Edit, and Delete Blog Posts
- Rich Text Editor for writing content
- Image Upload for blog posts
- Protected Routes — only post authors can edit/delete
- Responsive and clean UI

## 🛠️ Tech Stack

**Frontend:**
- React.js
- React Router DOM
- Axios
- MD Editor

**Backend:**
- Node.js
- Express.js
- MongoDB & Mongoose
- JSON Web Token (JWT)
- Multer (Image Upload)
- Bcryptjs (Password Encryption)
